Nyx: A character-driven AI Assistant

Nyx is a character-driven AI assistant that blends personality, natural language interaction, and tool usage into a comprehensive user companion.

Overview

Nyx is an experimental AI assistant designed to explore how conversational AI systems can become more context-aware and interactive. The assistant combines large language models with persistent memory, document retrieval, and external tool integrations to create a more capable and personalized assistant. The system allows users to upload documents, ask questions about their content, and interact with the assistant through a graphical interface. Nyx can also store information about the user, maintain conversational context, and access external tools such as weather services. Nyx is currently being developed as a Computer Science capstone project and will be presented at a university technology exhibition.

Features

Conversational AI

  • Interactive assistant with a consistent character-driven personality
  • Context-aware responses using conversation history
  • Dynamic relationship tracking based on user sentiment

Persistent Memory

  • Stores user preferences such as likes, dislikes, favorites, and name
  • Maintains conversation logs for long-term context
  • Memory retrieval used to personalize responses

Understanding

  • Upload and analyze TXT, PDF, and DOCX files
  • Document chunking and embedding generation
  • Retrieval-Augmented Generation for answering questions about uploaded documents

Tool Integrations

  • Weather information retrieval through external APIs
  • File reading and document processing tools
  • Designed for future expansion with additional capabilities

Graphical Interface

  • Desktop chat interface built with CustomTkinter
  • File upload support
  • Typing animation and conversational message layout
  • Toggle between local models and cloud-based models
